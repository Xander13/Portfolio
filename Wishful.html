00 Brief



In our 2021 Senior capstone, we were in a pandemic Covid-19 which made all in person interaction digital only.


RIT capstone is a senior "thesis" project that we have to create a product by end of semester. We were allowed to be part as teams or be individual contributor.

Wishful is a digital wishing well for people to get inspired during the hard time of Covid-19. We wanted to create a space for people to start thinking bright things they ever wish to imagine.





Our team contains 6 people, 2 were UX/UI designers, myself and Will were hybrid as being designer and developers, and lastly we had 2 developers.

Why the Jellyfish?

Based on several resources we have researched, we learned that Jellyfish have a symbolic connection to personal reflection. By making a wish, you’re giving life to a moment of self reflection, and putting your wish out there in hopes of better things to come.

We bound idea of limitless. Creating a interactive space that is the deep sea and deep reach of the sky to create the illusion of limitless. Thus allows us to create a refine color style to create calm and vibrant space. When selecting a jellyfish, it brightens up based on the corresponding wish tags it contains.












03
The Build



Being a hybrid Designer and Developer helps minimize gaps and communication barriers.

We only had two developers on the team, so Will and I helped them code out different areas of developments. I focused on coding a lot of the front-end UI development of the website and how the motions will work when users interacts. I have learned a lot of Javascript and using libraries such as three.js, jQuerry, and advance JS to help deploy our ideas into reality.

Things we learned:

Time Management

As realizing that we can dream all the stars we want to in Figma but we learned that not all thing can be me made well in scale at a specific time stamp.

We factor out how our project will set up based on our timeline. We set goals VMP and Sky High goals.

Performance

Creating 300+ Jellyfish and computing all that math power to a computer takes a lot of memory and graphics. We set parameters to populate number of jellyfish mesh baed on number of jellyfish in plane.

The results from that was thrilling to see all the jellyfish being populated and still runs really well.

Co-learning

Designer and developers have learned a lot from each other in the project process as understanding how things work with data algorithm, understanding 3d mesh through code and how compute them in real time.
