The Build



Being a hybrid Designer and Developer helps minimize gaps and communication barriers. We only had two developers on the team, so Will and I helped them code out different areas of developments. I focused on coding a lot of the front-end UI development of the website and how the motions will work when users interacts.



Things we learned:

Time Management


As realizing that we can dream all the stars we want to in Figma but we learned that not all thing can be me made well in scale at a specific time stamp.


We factored out how our project will set up based on our timeline. We set goals VMP and Sky High goals.

Performance


Creating 100+ Jellyfish and computing all that math power to a computer takes a lot of memory and graphics. We set parameters to populate number of jellyfish mesh baed on number of jellyfish in plane.


The results from that was thrilling to see all the jellyfish being populated and still runs really well.

Co-learning


Designer and developers have learned a lot from each other in the project process as understanding how things work with data algorithm, understanding 3d mesh through code and how compute them in real time.
