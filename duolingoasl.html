Duolingo ASL

Bringing non-varbal languages into Duolingo app experiance


Time: Currently work in progress



Role: Visual Design, Concpet



Devices: iOS App



Duolingo is a language learning app primarily focused on teaching spoken languages. It’s a fantastic platform for users to learn.


However, Duolingo overlooks a significant market: non-verbal language learning. We need to rethink how we approach teaching non-verbal languages.


Duolingo ASL aims to broaden this perspective, offering new ways to learn non-verbal languages and creating an inclusive, fun learning environment for everyone.
